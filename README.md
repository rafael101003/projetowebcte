# 🌐 Site Institucional — Centro Tecnológico Eduvale (Frontend)

## 📌 Descrição do Projeto1

Este projeto tem como objetivo desenvolver o **site institucional do Centro Tecnológico Eduvale**, contendo páginas informativas e módulos de divulgação acadêmica e social. O sistema será construído **exclusivamente em frontend**, simulando um ambiente real de entrega para uma instituição.

O foco principal do projeto é aplicar boas práticas de desenvolvimento web, priorizando:

- Acessibilidade
- Responsividade
- Usabilidade
- Organização de código
- Experiência do usuário (UX)

O projeto faz parte de uma atividade extensionista da disciplina de **Aplicações Web I**, com desenvolvimento colaborativo em equipe utilizando controle de versão.

---

## 🎯 Objetivos

- Desenvolver um site institucional completo em frontend
- Aplicar conceitos de HTML, CSS e JavaScript
- Implementar layout responsivo
- Garantir padrões mínimos de acessibilidade
- Simular fluxo profissional de desenvolvimento em equipe
- Utilizar versionamento com Git e workflow por branches

---

## 🧩 Módulos do Site

O site deverá conter as seguintes seções:

- Home
- Institucional
- Serviços
- Cursos
- Projetos
- Agenda
- Docentes
- Inclusão Digital
- Segurança
- Contato

---

## 🎨 Diretrizes Visuais

- Cores principais: **azul, amarelo e branco**
- Layout responsivo (desktop, tablet e mobile)
- Identidade visual consistente
- Navegação clara e intuitiva

---

## ♿ Acessibilidade

O projeto deve considerar:

- Uso correto de tags semânticas
- Textos alternativos em imagens
- Contraste adequado de cores
- Estrutura de títulos hierárquica
- Navegação compreensível

---

## 📱 Responsividade

O site deve funcionar adequadamente em:

- Desktop
- Tablet
- Smartphones

Uso recomendado de:

- Flexbox e/ou Grid
- Media queries
- Layout fluido

---

## 👥 Organização de Desenvolvimento

- Projeto desenvolvido em equipes
- Cada equipe responsável por módulos específicos
- Uso de branches por equipe
- Integração via branch de desenvolvimento
- Revisão antes de merge
